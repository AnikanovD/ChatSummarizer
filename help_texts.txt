[SUMMARIZE_HELP]
Команда /summarize

Использование: /summarize [начальная_дата] [конечная_дата]

Эта команда обобщает сообщения в чате за указанный период. Если даты не указаны, будет обобщен текущий день.

Формат дат: YYYY-MM-DD (год-месяц-день)

Примеры:
1. /summarize - обобщить сообщения за текущий день
2. /summarize 2023-06-01 - обобщить сообщения за 1 июня 2023 года
3. /summarize 2023-06-01 2023-06-30 - обобщить сообщения с 1 по 30 июня 2023 года

Бот проанализирует сообщения за указанный период и предоставит краткое обобщение основных тем и событий.

Примечание: Количество обрабатываемых сообщений для обобщения можно настроить с помощью команды /update_limit.

[ASK_HELP]
Команда /ask

Использование: /ask <ваш_вопрос>

Эта команда позволяет задать вопрос по истории чата. Бот проанализирует предыдущие сообщения и постарается дать ответ на ваш вопрос.

Пример:
/ask О чем говорили участники чата вчера?

Бот проанализирует историю чата и предоставит ответ, основываясь на содержании предыдущих сообщений.

Примечание: Количество сообщений, учитываемых при ответе на вопросы, можно настроить с помощью команды /update_limit.

[UPDATE_QUERY_LLM_HELP]
Команда /update_query_llm

Эта команда позволяет создать новую или обновить существующую функцию обработки естественного языка для текущего чата.

Процесс:
1. Введите команду /update_query_llm
2. Бот попросит вас ввести имя для новой функции. Имя не должно совпадать с существующими и не может быть 'default'.
3. После ввода имени, отправьте текстовый файл (.txt) с кодом новой функции.

Функция должна называться query_llm и принимать один аргумент (prompt). Она должна возвращать строку.

Пример содержимого файла:

def query_llm(prompt):
    # Ваш код здесь
    return "Ответ на запрос: " + prompt

Будьте осторожны при создании новых функций, так как они будут использоваться для обработки всех запросов в чате.

[UPDATE_PROMPT_HELP]
Команда /update_prompt

Использование:
1. Отправьте /update_prompt
2. Введите имя промпта, который вы хотите обновить (например, ASK_PROMPT, SUMMARIZE_PROMPT)
3. Введите новое содержание промпта

Эта команда позволяет настроить промпты, используемые для различных операций в текущем чате. Изменения будут сохранены и использованы для будущих операций только в этом чате.

[UPDATE_LIMIT_HELP]
Команда /update_limit

Использование:
1. Отправьте /update_limit
2. Введите имя лимита, который вы хотите обновить (например, summarize_limit, ask_limit, last_messages_limit)
3. Введите новое значение для лимита (должно быть положительное целое число)

Эта команда позволяет настроить количество загружаемых сообщений для различных операций в текущем чате. Изменения будут сохранены и использованы для будущих операций только в этом чате.
